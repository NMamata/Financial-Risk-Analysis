# Financial-Risk-Analysis
This project focuses on comprehensive financial analytics for a loan service provider, leveraging Power BI to uncover actionable insights into customer demographics, loan performance, and financial risk exposure.
The dashboard is designed through data transformation, DAX measures, and advanced visualizations, the project delivers a dynamic, interactive, and visually coherent solution for data-driven decision-making in the financial domain.
